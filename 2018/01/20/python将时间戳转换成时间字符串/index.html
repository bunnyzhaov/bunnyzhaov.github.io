python 将时间戳转换成时间字符串

2018-01-20 15:20:05 | python

时间字符串和时间戳的中间桥梁是时间元组,两者的转化必须通过时间元组

获取当前时间戳

把时间戳转化为时间元组用time.localtime(时间戳)

把时间元组转化为时间字符串用time.strftime('格式化字符串',时间元组)

time.strftime(‘格式化字符串’)默认取当前时间字符串
