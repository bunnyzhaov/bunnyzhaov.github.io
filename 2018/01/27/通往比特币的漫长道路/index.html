通往比特币的漫长道路

2018-01-27 15:24:46 | bitcoin比特币

发展顺序

  1. 传统金融
  2. 比特币
  3. 区块链
  4. 未来金融

通往比特币的漫长道路

  • 传统金融体系: 现金和信用卡
    • 网络信用卡: 身份信息等敏感信息传输，银行可以追查消费记录
    • 现金: 无需提交身份信息（确保了用户的匿名性），支持线下交易（无需从第三方获得交易批准）
  • 电子货币的发展历史
    • 哈希现金
    • B币
    • 比特黄金

其他对于电子货币探索失败的原因

  1. 依然中心机构(公司)发行
  2. 申请了专利保护,直接导致无法大面积推广
  3. 货币创造难度降低，数量激增，价值降低
  4. 没有得到广泛的认可和支持

传统金融体系的特点

  1. 绝对的中心化
    • 人民币由央行印刷和发行，记账（谁有多少钱），控制数量
    • QQ币由腾讯发布，Q币总量、交易、记账由腾讯完成
    • 传统的无论是法定货币，或者是虚拟代币，它都有特定的机构记账、审核、控制，也就是有中心点，来掌控一切
  2. 货币数量
    • 货币由中心机构发行，在不计后果的情况下，数量由发行机构控制，想发多少就发多少，想引多少就印多少
  3. 安全性
    • 如果中心节点被破坏，包括地震等不可抗力因素，会导致数据丢失，比如911事件导致金融数据丢失，损失不可估量.
    • 如果中心服务器被黑，比如腾讯被黑，或者公司倒闭，那么你的QQ币将何去何从.
    • 数据可被修改，比如阿里公司修改了你的账户余额和你所有的交易记录，你无法证明你的钱是你的钱.

比特币的诞生

比特币的诞生离不开密码学等学科的的发展。

  • 2008年8月，中本聪与开始可能对比特币感兴趣的人发送邮件讨论比特币
  • 2008年10月，公开白皮书，解释比特币协议，并公布了比特币源代码
  • 2010年11月，社区活跃，社区成员逐渐接手比特币项目，中本聪不再出现
  • 比特币从出生就定义为一种颠覆传统金融的货币，以货币而不是代币的身份来到世界

比特币的特点

  1. 去中心化:没有中心节点，它的记账由全球参与的计算机来完成
  2. 安全性：信息不可篡改（通过数字签名和哈希指针实现）
  3. 货币数量：比特币数量恒定，最高上限2100万枚，并且产生货币的难度会越来越大
  4. 分布式账本，意味着当交易发生时，区块链上的所有参与方都会在自己的账本上收到交易的信息，这些交易记录是完全公开，且经过加密、不可篡改的
