面试:我们到底需要什么

2018-07-14 22:01:22 | 生活与思考

因公司工作量增大,需要补充后端开发人员,初面工作由我来负责。虽然以前有过面试的经验,但
还是持谨慎态度,因为现在的环境下招到合适的人真的太重要了.

招聘需求:

  1. Python后端开发工程师
  2. 熟悉常用web框架: django,flask等
  3. 熟悉并熟练使用git和github
  4. 熟悉常用数据库: Mysql,Postgresql等
  5. 熟悉并熟练使用linux系统,可在linux环境下工作和开发
  6. 有1年以上工作经验
  7. 对互联网金融有所了解,有金融相关行业从业经验者优先

我面试的时候考察的几个方面:

  • 学习经历: 大学读的专业,何时又为何选择使用python
  • 离职原因,上一份工作的主要工作内容
  • 基本技术栈: 一般为python + django + mysql + git + github
  • 若写过爬虫,则聊聊爬虫的完整过程
  • git的常用命令,若条件允许手写否则口述:创建分支,提交代码,合并代码,申请pr等
  • linux的常用命令,若条件允许手写否则口述最常用的10-15个命令
  • 对技术和写代码是否有热情,是否真正的喜欢编程
  • 其他特别加分项
    • 有P2P公司工作经验
    • 有自己的技术博客并且在更
    • 熟悉并且一直科学上网,有问题不找度娘
  • 若技术水平一般达不到上述要求,但是对互联网金融了解并且对技术有极大的热情,也是可以的

总结

  • 或许有人会觉得我的面试比较业余。但是我认为符合我这几个方面的人不一定非常优秀,但是一定不会差。事实也会证明我的判断.
  • 从面试过程中来看,很是遗憾,这么简单的条件,现在的求职者当中符合我条件的不足30%.
  • 我们本可以做得更好不是么.
