    python 的logging处理之系统日志模块SysLogHandler

    2018-01-26 16:09:56

    python logging的组织模式

    1. logger 日志入口: 定义写入日志的内容和消息级别
    2. handler 处理方式: 定义如何处理logger传递来的日志记录, 比如是在控制台状态打印还是写入日志文件
    3. formatter 文本格式: 定义日志的文本组织方式
    4. filter 过滤器: 对logger传递给hanlder的消息进行额外的控制

    logging-handler-SysLogHandler模块

    python对日志处理方法SysLogHandler的定义

    参数:

    • address: 定义sysloghandler与远程unix主机通讯的soket(套接字),以(host,port)元组形式赋值,如果没有指定默认值(‘localhost’,514);
      • 可替代的方案是用字符串/dev/log代替(host,port)元组,建立本地soket发送消息到本机syslog
